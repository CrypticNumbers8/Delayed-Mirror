Delayed Mirror - Developed by Aniket Karna [My Portfolio]

Instructions on how to use this application:

  1. Select "Allow" when asked to use your cameras.(Full Screen Recommended)
  2. Select delay time(1-10), recommended setting = 5. (Real time = 1)
  3. Press spacebar or start button to begin recording.
  4. Recommend pressing spacebar and restarting recording every 60-90 sec.
  5. You can download a .webm file to save the recording on your computer.
